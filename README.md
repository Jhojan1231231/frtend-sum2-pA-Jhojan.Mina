# frtend-sum2-pA-Jhojan.Mina
"ViajarYa", una agencia de turismo ficticia especializada en paquetes de viaje nacionales e internacionales, cruceros, viajes 
de egresados y turismo de aventura.

"ViajarYa" opera desde hace diez anos ofreciendo experiencias de viaje personalizadas. Su sitio 
web debe transmitir confianza y facilitar que los visitantes descubran destinos, conozcan los 
servicios disponibles y puedan solicitar una reserva de forma sencilla.
